# ★ Созвездие класса ★

**Настольная игра о дружбе — теперь можно играть онлайн.**

Космическое путешествие по галактике дружбы, где на каждой звезде можно узнать
что-то новое друг о друге. Игра для **3–4 человек**, возраст **9+**, одна партия
≈ **20–30 минут**.

## 🌐 Играть онлайн

Откройте `index.html` в браузере (один файл, ничего устанавливать не нужно).

1. **Создать игру** — ведущий нажимает «Создать новую игру» и получает **код игры** в шапке.
2. **Поделиться** — код (или ссылку) копируют кнопкой 📋 и отправляют друзьям.
3. **Войти** — остальные открывают ту же страницу, вводят код и нажимают «Войти».
4. **Профиль** — каждый выбирает **имя**, **цвет фишки** и **космонавта** (выбор сохраняется в браузере на будущее).
5. **Жеребьёвка** — все бросают кубик; кто выбросил больше, ходит первым. При ничье — переброс.
6. Дальше ходят **по часовой стрелке** — всё по правилам игры.

> Сетевая часть работает на WebRTC (PeerJS) — отдельный сервер не нужен.
> Игра «живёт», пока открыта вкладка ведущего.

## 📖 Правила

### 🎯 Цель игры
Первым добраться от звезды **«Старт»** до большой звезды **«Созвездие дружбы»**,
выполняя весёлые задания по дороге. Но на самом деле выигрывают все — ведь к концу
игры вы узнаёте друг о друге много нового!

### 🚀 Ход игры по шагам
1. Все фишки ставят на клетку **«Старт»**. Жеребьёвкой решают, кто ходит первым.
2. Игрок бросает кубик и двигает фишку на столько звёзд вперёд.
3. На какой **цвет** звезды попал — берёт карточку такого же цвета и выполняет задание вслух.
4. После задания все аплодируют 👏 — правило обязательное и самое приятное!
5. Ход переходит к следующему игроку **по часовой стрелке**.
6. Кто первым дошёл до **«Созвездия дружбы»** — придумывает для команды общий девиз. Игра окончена!

### ✨ Золотое правило
В этой игре нельзя смеяться над ответами друг друга — можно только радоваться им.
Каждый ответ правильный, ведь он рассказывает о человеке что-то настоящее.

## 🌟 Игровое поле

«Галактика дружбы» — 16 звёзд между **Стартом** и **Созвездием дружбы**.
Цвет звезды задаёт тип задания:

| Цвет | Тип | Что делать |
|------|-----|-----------|
| 🩷 Розовый | **Комплимент** | Сказать другому что-то хорошее |
| 💙 Голубой | **Расскажи о себе** | Поделиться историей о себе |
| 💚 Зелёный | **Угадай** | Загадать или показать — остальные угадывают |
| 💜 Фиолетовый | **Вместе** | Командное задание для всех |
| 🧡 Оранжевый | **Доброе дело** | Придумать, как сделать добро |

## 🛠️ Технологии

Один HTML-файл — без сборки и зависимостей. Сеть — [PeerJS](https://peerjs.com/)
